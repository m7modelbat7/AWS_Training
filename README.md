# AWS Training Portal Scaffold

Production-ready Next.js starter scaffolded with:

- Next.js App Router + TypeScript
- Tailwind CSS with design tokens
- shadcn/ui-style reusable primitives
- Shared layouts for marketing and authenticated app sections
- Theme support with `next-themes`

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Project Structure

- `app/` route groups, pages, and global layout
- `components/layout/` shared layout components
- `components/theme/` theme provider and theme toggle
- `components/ui/` reusable UI primitives
- `lib/` utility helpers
